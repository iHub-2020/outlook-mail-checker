## English

**Remove ads and improve Outlook's web mailbox.**

### Features

- ✅ **Notification** when a new version is available.
- ✅ **Direct Access**: Two large buttons for direct access to Outlook or the calendar.
- ✅ **Multi-language Support**: English, Simplified Chinese.
- ✅ **Ad Removal**: Autohide first email AD (Please use [uBlock Origin](https://github.com/gorhill/uBlock) extension for better performance).
- ✅ **Customizable UI**:
    - Add/Remove the left banner containing quick links and Microsoft 365 content.
    - Add/Remove the Meet, Teams, and OneNote icons from the top toolbar.
    - Add/Remove Email counter (+ Color Support, RGB, RGBA, Hexa, Color Name).
    - Add/Remove Align Title Folder.
    - Add/Remove Support and rating button.
- ✅ **Theming**:
    - Add/Remove Background topbar theme url (Supports all image extensions such as jpg, png, gif etc. - please choose a 1920px width for better look).
    - Add/Remove Transparency topbar background.

### Installation

#### Google Chrome

**Install manually:**

1. Download the folder [google_unpacked](https://github.com/iHub-2020/Outlook-mail-checker/tree/main/google_unpacked).
2. In Chrome browser go to the Extensions page (or type in url `chrome://extensions/`).
3. Enable **Developer Mode** (toggle in the top right).
4. Click the **Load unpacked** button.
5. Select the directory `google_unpacked` from the downloaded git project.

## 中文

**移除广告并优化 Outlook 网页版邮箱体验。**

### 功能特性

- ✅ **更新通知**：新版本可用时发送通知。
- ✅ **快速访问**：提供两个大按钮，可直接访问 Outlook 邮箱或日历。
- ✅ **多语言支持**：支持英语和简体中文。
- ✅ **移除广告**：自动隐藏首封邮件广告（建议搭配 [uBlock Origin](https://github.com/gorhill/uBlock) 扩展使用以获得最佳效果）。
- ✅ **界面自定义**：
    - 添加/移除包含快速链接和 Microsoft 365 内容的左侧边栏。
    - 添加/移除顶部工具栏中的 Meet、Teams 和 OneNote 图标。
    - 添加/移除邮件计数器（支持自定义颜色：RGB, RGBA, 十六进制, 颜色名称）。
    - 添加/移除文件夹标题对齐功能。
    - 添加/移除“支持与评分”按钮。
- ✅ **主题美化**：
    - 添加/移除顶部栏背景主题 URL（支持 jpg, png, gif 等所有图片格式，建议选择宽度为 1920px 的图片以获得更好观感）。
    - 添加/移除顶部栏背景透明效果。

### 安装指南

#### Google Chrome (谷歌浏览器)

**手动安装：**

1. 下载 [google_unpacked](https://github.com/iHub-2020/Outlook-mail-checker/tree/main/google_unpacked) 文件夹。
2. 在 Chrome 浏览器中进入扩展程序页面（或在地址栏输入 `chrome://extensions/`）。
3. 开启右上角的 **“开发者模式” (Developer Mode)**。
4. 点击 **“加载已解压的扩展程序” (Load unpacked)** 按钮。
5. 选择刚才下载的项目中的 `google_unpacked` 目录。
