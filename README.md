### Installation

#### Google Chrome

**Install manually:**

1. Download the folder [google_unpacked](https://github.com/iHub-2020/Outlook-mail-checker/tree/main/google_unpacked).
2. In Chrome browser go to the Extensions page (or type in url `chrome://extensions/`).
3. Enable **Developer Mode** (toggle in the top right).
4. Click the **Load unpacked** button.
5. Select the directory `google_unpacked` from the downloaded git project.
